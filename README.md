# Anime Guestbook 🌸

An interactive, playful guestbook where visitors can sign in, see live stats, and click on an anime girl for cute reactions. Built with **React**, **Node.js**, and **PostgreSQL**.

---

## Features

- Cute anime girl with clickable reactions 🐱
- Falling petal animation ✨
- Live visitor stats: total visitors, countries, latest ID
- Responsive design for desktop and mobile
- Backend API for storing and fetching visitors
- PostgreSQL database integration

---

## Tech Stack

- **Frontend:** React, TypeScript, CSS animations
- **Backend:** Node.js, Express
- **Database:** PostgreSQL
- **Others:** CORS, dotenv for environment variables

---

## Getting Started

### Prerequisites

- Node.js >= 18
- PostgreSQL database
